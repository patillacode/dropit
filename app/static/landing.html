How it works

01

Get a token

Request access from the admin panel, or generate one via the API. Your token authenticates every upload and keeps your instance private.

02

Drop your file

Drag an HTML file into the drop zone, or POST it to the API. Choose how long you want it to stay live — from one hour to seven days.

03

Share the link

Your file is live instantly at a clean URL. When the TTL expires it disappears automatically — no manual cleanup, no orphaned files.

The API

One endpoint.
No SDK required.

A single POST /upload is all it takes. Bearer token in the header, file and TTL as form fields. The response hands you back a live URL.

Available TTLs

1h 6h 24h 48h 7d
bash
$ curl -X POST https://your.instance/upload \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -F "file=@page.html" \
    -F "ttl=24h"

  "url": "https://your.instance/f/abc123",
